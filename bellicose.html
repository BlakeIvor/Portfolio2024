Bellicose: Ext. Bellicose Ultimate Fighting

Contributions: State Machine Design and General Programming, Controller Support w/New Unity Input System (Unity, C#, Git, Github)

Bellicose: Ultimate fighting is a 2D Local Multiplayer Fighting Game created originally in 48 hours for IGDA Clark's February Frenzy 2024 Game Jam.

Bellicose was created by Blake Shea (Programming), Anthony Panasenko (2D Art), Jason Chen (Programming and UI), Sean McDowell (Sprites, Writing, Production), and Liam Quinn (Sound, UI)

Gameplay:

Each player can choose from a cast of characters (or random) and a map from the pool. They then start the game and fight to the death.

Each character runs off of a complex state machine to handle taking actions, getting hit, invincibility frames, and movement.
